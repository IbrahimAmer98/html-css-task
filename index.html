HTML (HyperText Markup Language) is the standard language for creating webpages.

It consists of various tags used to structure and display content.

Common HTML Tags:

  • <h1> to <h6> - Headings
  • <p> - Paragraph
  • <a> - Links
  • <img> - Images
  • <ul>, <ol>, <li> - Lists
  • <table> - Tables
  • <form>, <input>, <textarea> - Forms
